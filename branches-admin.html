Formula Reference
Fairness Score = BaseScore
              + MIN(TenureMonths × TenureBonusPerMonth, TenureMaxBonus)
              − TransfersReceived × TransferPenaltyPerMove
              − (RecentTransferPenalty if last transfer within RecentTransferWindow)
              − (OnLeavePenalty if currently on leave)

Higher score = higher priority in queue.
